# Proxy API 
a very basic API proxy to retrieve the data from the facebook GRAPH API.  
 
## Capabilities
written with **Python** using the *Flask* library
shows a hello at `:/`  
and a list of events at `:/events`  

## Hosting
hosted at *heroku* : https://proxy-ict-api.herokuapp.com/
